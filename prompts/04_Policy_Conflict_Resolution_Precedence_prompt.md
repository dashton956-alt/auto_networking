# Prompt: Policy Conflict Resolution & Precedence

Write a policy conflict detection and precedence resolution design for network intent governance.

Inputs:
- policy set (compliance, availability, performance, security)
- incoming intent object

Outputs:
- conflict report with involved policies
- resolved policy set
- precedence decisions and rationales

Requirements:
- include precedence model and tie-breaker algorithm
- include conflict mitigation recommendations
- include examples of conflicting policy cases
