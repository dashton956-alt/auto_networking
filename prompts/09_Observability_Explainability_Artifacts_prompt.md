# Prompt: Observability & Explainability Artifacts

Create an observability and explainability design for the decision pipeline.

Inputs:
- decision events (intent -> policy -> action -> outcome)
- telemetry and audit logs

Outputs:
- explainability API schema
- dashboard metrics and alert rules
- trace/event data model

Requirements:
- include immutable audit retention rules
- include "why" query API
- include compliance evidence collection
