# Prompt: Dark NOC Evolution Plan

Design a Dark NOC maturity model with explicit stage gates, metrics, and rollback criteria.

Inputs:
- current capability metrics (MTTR, scope, automation coverage)
- policy/oversight readiness

Outputs:
- stage definitions (0-4) with entry/exit criteria
- KQI/KPI metrics dashboard requirements
- gating and certification process

Requirements:
- include rollback conditions for stage regression
- include enforcement of compliance/audit requirements
- include stage transition playbooks
